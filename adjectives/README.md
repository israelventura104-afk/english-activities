# Adjectives Flashcards

Interactive A1–A2 adjective practice by **Israel Ventura**.

## Activities
1. **Flashcard deck** — 30 photo cards with flip, audio, and navigation  
2. **Memory match** — 8 pairs (photo ↔ word)  
3. **Speed round** — timed recognition  
4. **Emoji challenge** — guess the adjective from emojis  

## Run locally
Open `index.html` in your browser, or use GitHub Pages after deploy.

## GitHub Pages
Settings → Pages → Source: **Deploy from a branch** → `main` / root.
